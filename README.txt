===HELP GUIDE===

Stap 1: Start het script
Stap 2: Kies een optie uit het menu

Optie 1: Een nieuwe casus toevoegen

Wanneer er een nieuwe casus aangemaakt wordt, zal er een savefile gegenereerd worden.
Deze savefile heeft de extensie: '.BMMG'. Met dit bestand kunt u op elk gewenst moment verdergaan met een casus.
